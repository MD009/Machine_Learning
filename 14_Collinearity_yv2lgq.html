# Draw random samples from a normal (Gaussian) distribution
np.random.seed(1)
x1 = np.random.normal(size=100)
x2 = 0.5*x1 + np.random.randn(100)/10
y = 2+2*x1+0.3*x2+np.random.randn(100)

Form of the linear model is

  1. Y = beta0 + beta1 x1 + beta2 x2 + epsilon
  2. Y = 2 + 2 x1 +0.3 x2 + epsilon

Figure_1.axes[0].set_ylabel('Standardized Residuals',fontdict=axis_font);
